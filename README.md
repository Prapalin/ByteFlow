# ByteFlow AI 

ByteFlow AI is a modern AI-powered developer platform landing page built with **React** and **Tailwind CSS**. It provides a sleek, responsive, and interactive user interface designed to showcase AI-driven coding features like intelligent code completion, automated testing, and smart debugging.

## Features

-  Modern and responsive UI design
-  Built with React for dynamic component-based development
-  Tailwind CSS for fast and flexible styling
-  Interactive hero section with animated gradients
-  Code editor-style interface with syntax highlighting
-  Smooth animations and hover effects
-  Fully responsive across devices
-  Reusable React components

## Tech Stack

- **Frontend:** React.js
- **Styling:** Tailwind CSS
- **Icons:** Lucide React / React Icons
- **Syntax Highlighting:** React Syntax Highlighter
- **Build Tool:** Vite
